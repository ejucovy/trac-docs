announcer.subscribers.TicketCustomFieldSubscriber

In package TracAnnouncer

Allows users to subscribe to tickets that have their sid listed in any
field that has a name in the custom_cc_fields list.  The custom_cc_fields
list must be configured by the system administrator.

Implements:

  • announcer.api.IAnnouncementDefaultSubscriber
  • announcer.api.IAnnouncementSubscriber

Contains via Extension Points:

Options:

SectionNameTypeDefaultDocs
announcer custom_cc_fields ListOption Field names that contain users that should be notified on ticket changes
announcer always_notify_custom_cc_distributor ListOption email Comma seperated list of distributors to send the message to by default. ex. email, xmpp
announcer always_notify_custom_cc BoolOption true The always_notify_custom_cc will notify the users in the custom cc field by default when a ticket is modified.
