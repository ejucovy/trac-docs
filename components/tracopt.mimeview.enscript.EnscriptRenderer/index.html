tracopt.mimeview.enscript.EnscriptRenderer

In package Trac

Syntax highlighter using GNU Enscript.

Implements:

  • trac.mimeview.api.IHTMLPreviewRenderer

Contains via Extension Points:

Options:

SectionNameTypeDefaultDocs
mimeviewer enscript_modes ListOption text/x-dylan:dylan:4 List of additional MIME types known by Enscript. For each, a tuple `mimetype:mode:quality` has to be specified, where `mimetype` is the MIME type, `mode` is the corresponding Enscript mode to be used for the conversion and `quality` is the quality ratio associated to this conversion. That can also be used to override the default quality ratio used by the Enscript render, which is 2 (''since 0.10'').
mimeviewer enscript_path Option enscript Path to the Enscript executable.
