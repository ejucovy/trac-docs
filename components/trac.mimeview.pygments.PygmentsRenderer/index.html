trac.mimeview.pygments.PygmentsRenderer

In package Trac

HTML renderer for syntax highlighting based on Pygments.

Implements:

  • trac.env.ISystemInfoProvider
  • trac.mimeview.api.IHTMLPreviewRenderer
  • trac.prefs.api.IPreferencePanelProvider
  • trac.web.api.IRequestHandler

Contains via Extension Points:

Options:

SectionNameTypeDefaultDocs
mimeviewer pygments_default_style Option trac The default style to use for Pygments syntax highlighting.
mimeviewer pygments_modes ListOption List of additional MIME types known by Pygments. For each, a tuple `mimetype:mode:quality` has to be specified, where `mimetype` is the MIME type, `mode` is the corresponding Pygments mode to be used for the conversion and `quality` is the quality ratio associated to this conversion. That can also be used to override the default quality ratio used by the Pygments render.
