announcer.subscribers.JoinableGroupSubscriber

In package TracAnnouncer

Allows users to subscribe to groups as defined by the system
administrator.  Any ticket with the said group listed in the cc
field will trigger announcements to users in the group.

Implements:

  • announcer.api.IAnnouncementSubscriber
  • announcer.api.IAnnouncementPreferenceProvider

Options:

announcer joinable_groups ListOption []
