trac.ticket.default_workflow.ConfigurableTicketWorkflow

Ticket action controller which provides actions according to a workflow defined in trac.ini. The workflow is idefined in the `[ticket-workflow]` section of the [wiki:TracIni#ticket-workflow-section trac.ini] configuration file.

  • trac.env.IEnvironmentSetupParticipant
