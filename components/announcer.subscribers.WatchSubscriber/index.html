announcer.subscribers.WatchSubscriber

In package TracAnnouncer

Allows user to subscribe to ticket or wikinotification on a per
resource basis.  Watch, Unwatch links are added to wiki pages and tickets
that the user can select to start watching a resource.

Implements:

  • trac.web.api.IRequestFilter
  • trac.web.api.IRequestHandler
  • announcer.api.IAnnouncementSubscriber
  • trac.ticket.api.ITicketChangeListener
  • trac.wiki.api.IWikiChangeListener

Contains via Extension Points:

Options:

SectionNameTypeDefaultDocs
announcer ctxtnav_names ListOption Watch This, Unwatch This Text of context navigation entries. An empty list removes them from the context navigation bar.
announcer watchable_paths ListOption wiki/*,ticket/* List of URL paths to allow watching. Globs are supported.
