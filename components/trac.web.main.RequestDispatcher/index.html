trac.web.main.RequestDispatcher

In package Trac

Web request dispatcher.

This component dispatches incoming requests to registered
handlers.  Besides, it also takes care of user authentication and
request pre- and post-processing.

Contains via Extension Points:

  • trac.web.api.IAuthenticator
  • trac.web.api.IRequestHandler
  • trac.web.api.IRequestFilter
  • trac.web.api.IRequestHandler
